# CodeTiles

## For at køre client:
```bash
cd client
deno install
deno run build
deno run dev
```

## For at køre server:
#### Først skal deno install være kørt og client bygget
```bash
cd server
deno run dev
```


## API
kan læses på https://codetiles.voe.dk/docs.html
